== Installation ==
1. Download the LiteImage plugin from the WordPress Plugin Directory.
2. Upload the `liteimage` folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the WordPress admin panel (`Plugins > Installed Plugins`).
4. Configure settings at `Tools > LiteImage Settings` to enable/disable thumbnails or logging.
5. Use the `liteimage()` function in your theme or templates to display optimized images.
